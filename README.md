# Shaffira Kalam Arabic 💬

## Deskripsi Singkat
**Shaffira Kalam Arabic** adalah chatbot berbasis Artificial Intelligence (AI) yang dirancang khusus sebagai teman berlatih percakapan bahasa Arab. Chatbot ini bertindak sebagai tutor interaktif dan suportif untuk membantu pengguna meningkatkan kemampuan *Maharah Kalam* (Berbicara) dalam konteks sehari-hari.

## Maharah & Level yang Dipilih
* **Maharah:** Kalam (Berbicara/Percakapan)
* **Level Pengguna:** Pemula - Menengah
* **Alasan Pemilihan:** Maharah kalam seringkali menjadi tantangan karena minimnya partner atau teman untuk mempraktikkan bahasa Arab secara langsung. Shaffira hadir sebagai solusi *partner muhadatsah* yang tidak menghakimi, siap mengoreksi dengan ramah, dan memotivasi pengguna untuk berani berekspresi.

## Teknologi & API yang Digunakan
* **Bahasa Pemrograman:** Python 3.8+
* **AI API Service:** Google Gemini API (`gemini-1.5-flash`)
* **Libraries:** `google-generativeai`, `python-dotenv`, `rich` (untuk UI Terminal).

## Fitur Utama
1. **Persona Shaffira:** Karakter tutor yang ceria, interaktif, dan langsung mengoreksi tata bahasa secara konstruktif.
2. **Riwayat Percakapan (Memory):** Chatbot mengingat alur percakapan selama satu sesi berlangsung.
3. **Tiga Mode Pembelajaran:**
   - *Ta'aruf* (Perkenalan Diri)
   - *Maukif* (Bermain Peran/Roleplay di Restoran)
   - *Hiwar Bebas* (Ngobrol Santai)
4. **Terminal UI yang Menarik:** Menggunakan library `rich` agar pembacaan teks di terminal lebih rapi dan berwarna.
